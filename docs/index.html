Features

Everything your shell
config actually needs

wayu replaces the scattered, manual editing of shell config files with structured commands — each with validation, dry-run preview, and automatic backups.

🛤️

Array-Based PATH Management

A centralized WAYU_PATHS=() array with a single export loop. Deduplication, validation, and a clean command that removes entries pointing to missing directories. Fixes the silent bug where new paths added after exports were completely ineffective.

v3.0.0 — Breaking change
🔌

Plugin Management

54-plugin registry. Install from any Git URL. Dependency resolution with cycle detection. Conflict detection across env vars, functions, and aliases. Priority-based load ordering.

💾

Auto Backup & Restore

Every modification creates a timestamped backup automatically. Keeps the last 5. Restore any config type with one command. You cannot accidentally lose your shell config.

🐚

Multi-Shell

Automatic Bash/Zsh detection from $SHELL. Shell-specific config files with correct syntax. Migrate between shells with wayu migrate.

🔍

Dry-Run Mode

The --dry-run flag previews any change before it's applied. Works across every command. See exactly what would change before it happens.

📜

Scriptable CLI

BSD sysexits.h exit codes (0, 1, 64–78). Fully non-interactive — no prompts, no fuzzy finders. Safe for pipes, CI/CD, and provisioning scripts. --yes flag for confirmations.

✅

Input Validation

Shell identifier validation, reserved word checking (if, export, etc.), dangerous character escaping, and path sanitization.

🔗

External Alias Sources

Surface read-only aliases from external tools (e.g. fabric patterns) in wayu alias list via alias-sources.conf.

Commands

One syntax.
Every config.

Every command follows the same pattern: wayu <command> <action> [args]. No flags to memorize. No config files to learn.

path management 
# Add a new PATH entry
wayu path add ~/.cargo/bin

# List all entries
wayu path list

# Remove an entry
wayu path rm /usr/local/bin

# Remove broken entries (preview first)
wayu path clean --dry-run
wayu path clean --yes

# Remove duplicates
wayu path dedup --yes
alias management 
# Add aliases
wayu alias add ll 'ls -la'
wayu alias add gs 'git status'
wayu alias add gc 'git commit'

# List all (includes external sources)
wayu alias list

# Remove an alias
wayu alias rm ll

# Add constants
wayu constants add EDITOR nvim
plugin management 
# Browse 54 popular plugins
wayu plugin search
wayu plugin search syntax

# Install from registry or URL
wayu plugin add zsh-autosuggestions
wayu plugin add https://github.com/user/plugin.git

# Manage plugins
wayu plugin list
wayu plugin enable zsh-autosuggestions
wayu plugin priority zsh-autosuggestions 50
wayu plugin update --all
backup & restore 
# Backups are created automatically.
# These commands let you manage them.

# List all backups
wayu backup list

# List backups for one config type
wayu backup list path

# Restore most recent backup
wayu backup restore path

# Clean up old backups (keeps last 5)
wayu backup rm

Two Modes

Built for scripts.
Built for humans.

$ CLI Mode

Non-interactive.
Pipe-safe. Scriptable.

The default mode. Zero prompts, zero interactive elements. Designed for shell scripts, CI/CD pipelines, and provisioning tools.

  • BSD sysexits.h exit codes (0, 1, 64–78)
  • No fuzzy finders, no spinners, no TTY required
  • --yes flag for confirmation-free destructive ops
  • Fails fast on errors with categorized messages
  • Safe to use in if statements and pipes
⬛ TUI Mode

Full-screen.
Visual. Interactive.

Launch with wayu --tui. A complete terminal UI built from scratch — no external TUI libraries.

  • Vim-style navigation (j/k, arrow keys)
  • 8 views: PATH, Aliases, Constants, Plugins, Backups…
  • Fuzzy search and inline filtering
  • Confirmation modals before destructive actions
  • Alternate screen buffer — terminal history preserved

Config files created in ~/.config/wayu/

init.{zsh,bash}
Main orchestrator. Source this in your shell RC file — it loads everything else in the correct order.
path.{zsh,bash}
Centralized WAYU_PATHS=() array with single export loop. All your PATH entries, ordered and deduplicated.
aliases.{zsh,bash}
All alias definitions. Validated on write. Backed up before every change.
constants.{zsh,bash}
Environment variable exports. Warns about lowercase names (convention: UPPER_CASE).
alias-sources.conf
Shell-agnostic. Surface read-only aliases from external tools like fabric patterns.
backup/
Timestamped backups. Created automatically before every modification. Keeps last 5.
